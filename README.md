# Functional Prime

Examples of partial application, currying, uncurrying, and composition in Scala, JavaScript, Python, Java 8.

## To run Scala

1. `cd scalaFP/`
1. `(scalac src/main/scala/Main.scala && scala Main)`

## To run JavaScript

1. `cd jsFP/`
1. `npm start`

## To run Python

1. `cd pythonFP/`
1. `python script.py`

## To run Java

1. `cd javaFP/`
1. `(cd src/ && javac Main.java && java Main)`
